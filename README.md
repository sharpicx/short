## installation
ensure that your PATH have been set.
```shell
~$ export PATH=$HOME/.local/bin:$PATH
```
and then, save this script to the PATH.
```shell
~$ curl -s https://raw.githubusercontent.com/sharpicx/short/main/short -o ~/.local/bin/short
```
